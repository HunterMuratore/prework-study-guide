Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS is an external stylesheet read by your HTML. It uses property/value pairs and the appropriate HTML selectors to change the style of your HTML
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b [branch-name]: creates a new branch and switches to it
  • git add -A: add the changed files to the staging area of Git
  • git commit -m [message]: commit the changes to our repo with a descript message
  • git pull origin main: make sure current local branch is up to date with the remote branch in GitHub
  • git push origin [branch-name]: push local branch to the remote branch
  • Once work on the feature branch is complete create a pull request on GitHub to merge the changes into the main branch
